# Проект: Movie Reviews API

## Цель

- Создать REST API для платформы рецензий на фильмы.
- Пользователи могут регистрироваться, авторизоваться, просматривать фильмы и рецензии. Авторизованные пользователи могут писать рецензии, ставить лайки и управлять своими записями.
- Модераторы могут управлять контентом, администратор — всей платформой.
- Необходимо внутри репозитория создать Bruno коллекцию, описывающую все эндпоинты API.

_Хостить работу не надо_

Перед сдачей работы создать:
- 5 фильмов в БД
- 3 жанра и привязать к фильмам (связь многие-ко-многим)
- 5 рецензий с лайками
- админа с кредами `admin:qwerty123`
- модератора `moderator:qwerty123`
- юзера `user:qwerty123`

---

## Основные сущности

### 1. `User`

| Поле        | Тип    | Описание                                                   |
| ----------- | ------ | ---------------------------------------------------------- |
| `id`        | number | Уникальный идентификатор                                   |
| `username`  | string | Имя пользователя                                           |
| `email`     | string | Email (уникальный)                                         |
| `password`  | string | Хэшированный пароль                                        |
| `role`      | string | Роль: `"user"`, `"moderator"`, `"admin"`                   |
| `bio`       | string | Краткая биография (опционально)                            |
| `createdAt` | date   | Дата регистрации                                           |

---

### 2. `Movie`

| Поле          | Тип                   | Описание                          |
| ------------- | --------------------- | --------------------------------- |
| `id`          | number                | Уникальный идентификатор          |
| `title`       | string                | Название фильма                   |
| `director`    | string                | Режиссёр                          |
| `year`        | number                | Год выпуска                       |
| `country`     | string                | Страна производства               |
| `description` | string                | Описание                          |
| `posterUrl`   | string                | Ссылка на постер (опционально)    |
| `createdBy`   | number (FK → User.id) | Кто добавил                       |
| `createdAt`   | date                  | Дата добавления                   |

---

### 3. `Genre`

| Поле   | Тип    | Описание                 |
| ------ | ------ | ------------------------ |
| `id`   | number | Уникальный идентификатор |
| `name` | string | Название жанра (уникальное) |

> Связь `Movie ↔ Genre` — многие-ко-многим через таблицу `MovieGenre`.

---

### 4. `Review`

| Поле        | Тип                    | Описание                             |
| ----------- | ---------------------- | ------------------------------------ |
| `id`        | number                 | Уникальный идентификатор             |
| `movieId`   | number (FK → Movie.id) | Фильм                                |
| `userId`    | number (FK → User.id)  | Автор рецензии                       |
| `rating`    | number (1–10)          | Оценка                               |
| `title`     | string                 | Заголовок рецензии                   |
| `body`      | string                 | Текст рецензии                       |
| `status`    | string                 | `"pending"`, `"approved"`, `"rejected"` |
| `createdAt` | date                   | Дата публикации                      |

> Рецензия при создании получает статус `"pending"` и становится публично видна только после перевода в `"approved"` модератором или администратором.

---

### 5. `Like`

| Поле       | Тип                    | Описание                  |
| ---------- | ---------------------- | ------------------------- |
| `id`       | number                 | Уникальный идентификатор  |
| `reviewId` | number (FK → Review.id)| Рецензия                  |
| `userId`   | number (FK → User.id)  | Кто поставил лайк         |
| `createdAt`| date                   | Дата                      |

> Один пользователь может поставить лайк одной рецензии только один раз (уникальная пара `reviewId + userId`).

---

## Аутентификация

Использовать JWT (JSON Web Token).  
Токен выдаётся при логине и передаётся в заголовке:  
`Authorization: Bearer <token>`

---

## Эндпоинты

### Auth

| Метод  | URL                   | Описание                               | Доступ         |
| ------ | --------------------- | -------------------------------------- | -------------- |
| `POST` | `/api/auth/register`  | Регистрация нового пользователя        | Публично       |
| `POST` | `/api/auth/login`     | Вход, получение JWT                    | Публично       |
| `GET`  | `/api/auth/profile`   | Данные текущего пользователя           | Авторизованные |
| `PUT`  | `/api/auth/profile`   | Обновить `username` и `bio`            | Авторизованные |

---

### Movies

| Метод    | URL               | Описание                                                                 | Доступ         |
| -------- | ----------------- | ------------------------------------------------------------------------ | -------------- |
| `GET`    | `/api/movies`     | Список фильмов (фильтрация по жанру, режиссёру, году; сортировка; пагинация) | Публично       |
| `GET`    | `/api/movies/:id` | Фильм по ID с жанрами и одобренными рецензиями                          | Публично       |
| `POST`   | `/api/movies`     | Добавить фильм                                                           | Авторизованные |
| `PUT`    | `/api/movies/:id` | Обновить фильм (только автор или модератор/админ)                        | Авторизованные |
| `DELETE` | `/api/movies/:id` | Удалить фильм (только модератор или админ)                               | Модератор/Админ |

---

### Genres

| Метод    | URL                | Описание                              | Доступ          |
| -------- | ------------------ | ------------------------------------- | --------------- |
| `GET`    | `/api/genres`      | Список всех жанров                    | Публично        |
| `POST`   | `/api/genres`      | Создать жанр                          | Модератор/Админ |
| `DELETE` | `/api/genres/:id`  | Удалить жанр                          | Только админ    |

---

### Reviews

| Метод    | URL                           | Описание                                           | Доступ          |
| -------- | ----------------------------- | -------------------------------------------------- | --------------- |
| `POST`   | `/api/movies/:id/reviews`     | Добавить рецензию (статус `pending`)               | Авторизованные  |
| `GET`    | `/api/movies/:id/reviews`     | Одобренные рецензии к фильму                       | Публично        |
| `DELETE` | `/api/reviews/:id`            | Удалить свою рецензию (или модератор/админ)        | Авторизованные  |
| `PATCH`  | `/api/reviews/:id/status`     | Изменить статус рецензии (`approved`/`rejected`)   | Модератор/Админ |

---

### Likes

| Метод    | URL                        | Описание                            | Доступ         |
| -------- | -------------------------- | ----------------------------------- | -------------- |
| `POST`   | `/api/reviews/:id/like`    | Поставить лайк рецензии             | Авторизованные |
| `DELETE` | `/api/reviews/:id/like`    | Убрать лайк                         | Авторизованные |

---

### Admin

| Метод    | URL                       | Описание                                  | Доступ       |
| -------- | ------------------------- | ----------------------------------------- | ------------ |
| `GET`    | `/api/admin/users`        | Список всех пользователей                 | Только админ |
| `PATCH`  | `/api/admin/users/:id/role` | Изменить роль пользователя              | Только админ |
| `DELETE` | `/api/admin/users/:id`    | Удалить пользователя                      | Только админ |
| `GET`    | `/api/admin/reviews/pending` | Список рецензий на модерации           | Модератор/Админ |
